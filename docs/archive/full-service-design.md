**CONFIDENTIAL**

## 1. 서비스 정의 및 핵심 가치 (Overview)
ALLBLUE(올블루)는 파편화된 온라인 패션 쇼핑몰의 단품 데이터를 수집하여, AI가 새로운 코디 조합(룩북)을 자동 생성하고 사용자에게 통합된 쇼핑 경험을 제공하는 **AI 패션 오케스트레이션 플랫폼**입니다.

| 항목 | 내용 |
| :--- | :--- |
| **서비스명** | ALLBLUE (올블루) |
| **슬로건** | 파편화된 쇼핑몰 데이터를 하나로, AI 룩북으로 완성하다 |
| **핵심 가치** | AI 기반의 창의적인 코디 조합(Orchestration) 제안 |
| **타겟 유저** | 코디 조합에 고민이 많은 남녀 온라인 쇼핑 유저 |
| **핵심 엔진** | Google Gemini (Nano Banana) API 기반 가상 피팅 및 조합 |

---

## 2. 서비스 차별화 포인트 (USP)
기존 패션 어그리게이터(지그재그, 에이블리 등)가 '상품 모아보기'에 집중했다면, ALLBLUE는 **'AI를 통한 능동적 조합'**에 집중합니다.

1. **상품 나열 vs AI 룩북:** 단순 상품 리스트가 아닌, AI가 제안하는 완성된 스타일링(룩북) 중심의 탐색.
2. **개별 이동 vs 통합 결제:** 쇼핑몰별 개별 결제의 불편함을 해소하고, 카페24 연동을 통한 통합 결제 지향.
3. **사이즈 표기 vs Fit Score:** 단순 수치 제공을 넘어 사용자의 체형 대비 적합도를 점수로 환산하여 제안.

## 3. 핵심 서비스 플로우 (Service Flow)

### 3-1. 자동 갤러리 모드 (Auto Gallery)
AI가 수집된 데이터를 바탕으로 자동 코디를 생성하여 사용자에게 추천하는 방식입니다.
- **Step 1:** AI 엔진이 스타일, 색상, 시즌을 고려하여 최적의 조합 생성.
- **Step 2:** Gemini API를 통해 모델 착용 이미지(룩북) 합성.
- **Step 3:** 사용자는 갤러리에서 룩북을 탐색하고 마음에 드는 상품을 클릭하여 원본몰로 이동.

## 4. 수익 모델 (Business Model)
| 단계 | 모델명 | 상세 내용 |
| :--- | :--- | :--- |
| **1순위** | 어필리에이트 | 룩북 내 상품 클릭 후 구매 발생 시 거래 수수료 취득 (MVP) |
| **2순위** | 셀러용 SaaS | 카페24 입점사 대상 AI 룩북 노출 및 분석 리포트 구독 |
| **3순위** | 광고 모델 | 특정 브랜드 상품의 코디 매칭 우선순위 부여 및 노출 광고 |
| **4순위** | 데이터 판매 | 트렌드 매칭 데이터 및 유저 선호도 리포트 판매 |

## 5. 기술 스택 (Technical Stack)
- **Backend:** Java 21, Spring Boot 3.5
- **Frontend:** Next.js (React), TailwindCSS
- **AI:** Google Gemini API, n8n Worker
- **Crawler:** Python (Playwright, curl_cffi)
- **Infrastructure:** AWS (S3, RDS, EC2), Redis
- **Monitoring:** Prometheus, Grafana, Loki

## 6. 데이터베이스 및 도메인 설계 방향
기존 '카드(Card)' 중심 구조에서 **'룩북(Lookbook)'과 '상품(Product)'** 중심 구조로 전면 개편합니다.

- **Lookbook:** AI가 생성한 코디의 마스터 데이터 (이미지, 스타일, 상태값).
- **Product:** 독립적인 핵심 상품 데이터 (브랜드, 가격, 카테고리, 원본 URL).
- **LookbookItem:** 룩북과 상품을 연결하는 매핑 데이터 (상의/하의/신발 구분).
- **Seller:** 카페24 등 입점 쇼핑몰 정보 및 API 연동 키 관리.
- **FitProfile:** 사용자 체형 데이터 (키, 몸무게, 어깨너비 등).
