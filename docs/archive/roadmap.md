# ALLBLUE 개발 로드맵 (Development Roadmap)

## Phase 1: 데이터 수집 및 AI PoC (4주)
**목표:** 상품 데이터 확보 및 AI 룩북 생성 기술 검증.
- 백엔드 리팩토링: `Lookbook`, `Product` 엔티티 구조 구축.
- Gemini API 연동을 통한 비동기 룩북 생성 파이프라인 PoC.
- SNS(Instagram)를 통한 생성된 코디 콘텐츠 시장성 테스트.

## Phase 2: 웹 서비스 MVP 구축 (4주)
**목표:** 실제 사용자가 사용 가능한 플랫폼의 최소 기능 구현.
- Next.js 기반 갤러리 웹 사이트 구축.
- 자동 생성 룩북 피드 및 무한 스크롤 구현.
- Redis 캐싱을 통한 성능 최적화.

## Phase 3: 카페24 연동 및 통합 결제 (4주)
**목표:** 실제 수익 창출이 가능한 쇼핑 연동 기능 도입.
- 카페24 공식 API를 통한 상품/재고/주문 데이터 연동.
- PortOne 결제 모듈 탑재 및 통합 결제 프로세스 구현.
- 주문 처리를 위한 Saga 패턴 기반의 분산 트랜잭션 구현.

## Phase 4: 확장 및 고도화 (진행중)
**목표:** 플랫폼 성장 및 데이터 기반 수익화 강화.
- 고도화된 추천 엔진 탑재 (유저 취향 데이터 기반).
- 셀러 전용 대시보드 및 분석 리포트 SaaS 베타 출시.
- 글로벌 진출을 위한 다국어 및 통화 지원.
