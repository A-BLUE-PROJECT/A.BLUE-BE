# ALLBLUE 서비스 정의 및 핵심 가치 (Service Overview)

## 1. 서비스 정의
**ALLBLUE (올블루)**는 파편화된 온라인 패션 쇼핑몰의 단품 데이터를 수집하여, **AI(Gemini)가 최적의 코디 조합(룩북)을 자동으로 생성**하고 사용자에게 맞춤형 쇼핑 경험을 제공하는 **AI 패션 오케스트레이션 플랫폼**입니다.

## 2. 핵심 가치 (Core Value)
- **단순 모음에서 조합으로:** 기존 서비스(지그재그, 에이블리 등)가 상품을 '모아 보여주는(Aggregation)' 것에 집중했다면, ALLBLUE는 AI를 통해 여러 브랜드의 옷을 '조합(Orchestration)'하여 새로운 가치를 창출합니다.
- **코디 중심 탐색:** 사용자는 개별 상품이 아닌, 완성된 '룩북'을 통해 쇼핑 영감을 얻습니다.
- **체형 기반 핏 점수(Fit Score):** 사용자의 체형 데이터와 상품 실측 데이터를 비교하여 AI가 추천 신뢰도를 높입니다.

## 3. 주요 서비스 플로우
### 3-1. 자동 갤러리 모드 (Auto Gallery)
- 시스템이 무신사 등에서 수집한 상품(상의, 하의, 신발) 데이터를 바탕으로 AI가 자동 코디를 생성.
- 사용자는 틴더 방식의 스와이프나 무한 스크롤로 룩북을 탐색.
- 마음에 드는 상품 클릭 시 원본 쇼핑몰로 딥링크 이동.

### 3-2. 유저 커스텀 모드 (User Custom Mode)
- 사용자가 원하는 카테고리별 상품을 직접 선택.
- AI가 해당 조합의 모델 착용 이미지(합성)와 핏 점수를 즉시 생성.

## 4. 수익 모델 (Business Model)
1. **어필리에이트 (Affiliate):** 외부 쇼핑몰 구매 연결 시 발생하는 판매 수수료.
2. **셀러용 SaaS 구독:** 카페24 입점사 등에게 AI 룩북 생성 및 분석 리포트 제공.
3. **광고 상품:** 특정 브랜드 상품의 코디 매칭 우선순위 부여.
